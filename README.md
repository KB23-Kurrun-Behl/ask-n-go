📌 About Ask N' Go

▪️This application was developed as part of my volunteering experience.
▪️The goal of this project is to answer technology-centered questions to make required tasks easy to understand and complete.

🎯 Features

▪️A simple user-interface offering quick and easy-to-use navigation across the application's webpages
▪️The application is browser-based meaning no installation is required
▪️The application contains hundreds of documents that are designed to provide step-by-step instructions to complete a particular task

⚒️ Tech Stack

▪️HTML
▪️CSS
▪️JavaScript
